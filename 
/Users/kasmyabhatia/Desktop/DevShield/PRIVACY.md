
# DevShield - Privacy Statement

## Privacy Policy

DevShield is designed with privacy as a core principle. This document explains how the extension handles user data.

---

## Data Collection

**DevShield does NOT collect, store, or transmit any user data.**

All analysis performed by this extension is done locally within the user's browser. No information is sent to external servers.

---

## Local-Only Analysis

The extension performs the following analysis entirely locally:

1. **DOM Inspection**: Analyzing HTML structure (forms, inputs, scripts, images)
2. **Security Patterns**: Scanning for common web security patterns
3. **Score Calculation**: Computing health scores based on local analysis

---

## No External Communication

- ❌ No data is sent to any server
- ❌ No analytics or tracking
- ❌ No third-party services
- ❌ No cloud storage
- ❌ Works completely offline after installation

---

## Permissions Used

| Permission | Purpose |
|------------|---------|
| `activeTab` | Access current tab URL for display |
| `scripting` | Execute analysis script on webpage |

These permissions are used solely for the extension's core functionality and do not enable data collection.

---

## Changes to This Policy

If this policy changes, the extension will be updated on GitHub. The extension will never add data collection without explicit user notification.
